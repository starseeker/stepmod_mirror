  • application object;

  • conformance.

3.1.3 Other terms and definitions

For the purposes of this document, the following terms and definitions apply:

3.1.3.1
design phase

period during which the engineering representation of a product is dynamic

3.1.3.2
mechanical part

physical object which can be formed from material into a static shape

3.1.3.3
saved view

stored and retrievable specific orientation and magnification factor of a model

3.1.3.4
solid model

complete 3D representation of the nominal shape of a product so that any point in space can be determined to be inside, on the boundary, or outside the solid model

3.1.3.5
subassembly

constituent that is an assembly

3.1.3.6
supplemental geometry

geometric elements included in product definition data to communicate design requirements but not intended to represent a portion of the manufactured product
