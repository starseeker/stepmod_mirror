Annex G
(informative)

Computer interpretable listings

This annex provides a listing of the complete EXPRESS schema referenced in Annex A of this part of ISO 10303 without comments or other explanatory text. It also provides a listing of the EXPRESS entity names and corresponding short names as specified in Annex B of this part of ISO 10303. The content of this annex is available in computer-interpretable form and can be found in Table G.1 or at the following URLs:

   Short names: http://www.tc184-sc4.org/Short_Names/
   EXPRESS: http://www.tc184-sc4.org/EXPRESS/
