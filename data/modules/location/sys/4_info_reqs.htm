This clause constitutes the Application Reference Model of the application module.

NOTE 1  A graphical representation of the information requirements is given in Annex C.

NOTE 2  The mapping specification is specified in 5.1. It shows how the information requirements are met, using common resources and constructs defined or imported in the MIM schema of this application module.
