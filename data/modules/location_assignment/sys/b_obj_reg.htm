Annex B
(normative)

Information object registration

B.1 Document Identification

To provide for unambiguous identification of an information object in an open system, the object identifier

{ iso standard 10303 part(xxxx) version(1) }

is assigned to this part of ISO 10303. The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.

B.2 Schema identification

B.2.1 Location_assignment_arm schema identification

To provide for unambiguous identification of the schema specifications given in this application module in an open information system, the object identifiers are assigned as follows:

{ iso standard 10303 part(xxxx) version(1) schema(1) location-assignment-arm(1) }

is assigned to the Location_assignment_arm schema. The meaning of this value is defined in ISO 8824-1, and is described in ISO 10303-1.

B.2.2 Location_assignment_mim schema identification

To provide for unambiguous identification of the schema specifications given in this application module in an open information system, the object identifiers are assigned as follows:

{ iso standard 10303 part(xxxx) version(1) schema(1) location-assignment-mim(2) }

is assigned to the Location_assignment_mim schema. The meaning of this value is defined in ISO 8824-1, and is described in ISO 10303-1.
