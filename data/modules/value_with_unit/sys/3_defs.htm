3 Terms, definitions and abbreviations

3.1 Terms defined in ISO 10303-1:1994

For the purposes of this part of ISO 10303, the following terms defined in ISO 10303-1:1994 apply:
  • application;
  • application object;
  • application protocol;
  • application reference model;
  • data;
  • information;
  • integrated resource;
  • product;
  • product data;
  • unit of functionality.

3.2 Terms defined in ISO 10303-202:1997

For the purposes of this part of ISO 10303, the following terms defined in ISO 10303-202:1997 apply:
  • application interpreted construct.

3.3 Terms defined in ISO/TS 10303-1001:2001

For the purposes of this part of ISO 10303, the following terms defined in ISO/TS 10303-1001:2001 apply:
  • application module;
  • module interpreted module.

3.4 Abbreviations

For the purposes of this part of ISO 10303, the following abbreviations apply:

AM application module
ARM application reference model
MIM module interpreted module
UoF unit of functionality
URL universal resource locator
