Interface IXdmEnumerator 

public interface IXdmEnumerator
implements IEnumerator

This class represents an enumeration of the values in an XPath sequence. It implements the IEnumerator interface, and the objects returned are always instances of XPathItem. In addition to the methods defined by IEnumerator, an additional method GetAnother must be implemented: this provides a new iterator over the same sequence of items, positioned at the start of the sequence.

Because the underlying value can be evaluated lazily, it is possible for exceptions to occur as the sequence is being read.
