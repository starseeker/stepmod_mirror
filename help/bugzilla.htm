Bugzilla location

A Bugzilla (http://www.bugzilla.org) instance is used to manage and track the issues raised against the STEPmod documents. The Bugzilla for ISO 10303 is at: http://http://www.wikistep.org/bugzilla/. This instance is tailored for ISO 10303 usage.

Bugzilla help location

Help for use of the 10303 instance of Bugzilla in development is at: http://www.wikistep.org/index.php/ISO_10303_usage_of_Bugzilla. http://www.bugzilla.org
