Instructions for accessing STEP Modules

If you are not a STEPmod developer and want to access and keep up to date with the STEP modules then read on.

If you want to contribute to the development of the STEP modules, then you need to be a developer on SourceForge and set up a secure shell.

The modules are being developed as a project on SourceForge:

  • SourceForge: https://sourceforge.net/
  • STEPmod project on SourceForge: http://stepmod.sourceforge.net/

All the modules are being developed in XML and managed under CVS at SourceForge. Therefore to browse the modules, you will need:

  • a browser with a built in XSL parser (Internet Explorer v6 works) to view the modules;
  • a CVS client to download an up to date set of the modules (winCVS works).

Setting up winCVS

Be aware that this will only let you download you will not be able to upload any changes. You will need to be a Sourceforge developer for that.
  1. Install winCVS version 1.2 from:
    https://sourceforge.net/project/showfiles.php?group_id=10072
  2. When starting it will ask you for the CVSroot. Enter:
    :pserver:anonymous@cvs.sourceforge.net:/cvsroot/stepmod

    It may also ask for authentication, use "passwd" on CVS server (select this from the pull down options) and version - use cvs1.10 Standard

    You can check these in: winCVS Admin->Preferences

  3. Now download the entire modules repository.
    Create a directory in which to store the modules.
    In winCVS

    Admin->Command Line

    Select "Click to specifying which local directory the command applies"

    Select the directory you have just created.
    Enter:
    cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/stepmod co stepmod
    as the CVS command line
    Press OK

    winCVS should start downloading the repository into your directory.

  4. Once you have downloaded the repository, you should be able to update it on a daily basis.

    First make CVS remember the modules directory.

    View-> browse location ->Change

    Select the directory

    Now save the settings
    View-> browse location ->Save Settings

    CVS should always open into this directory.

Browsing the modules

To browse the modules, open:

stepmod/nav/index.ml

in a browser with a built in XSL parser such as Internet Explorer 6.
