Module Creation

The following stages are required in the creation of a module:
  • Create the directory and files for the module using the JScript mkmodule.
  • Add the new module to the index of modules: repository_index.xml
  • Create the ARM, the MIM short and long form EXPRESS files storing them in the module directory.
    The ARM EXPRESS should be called arm.exp,
    the MIM short form EXPRESS should be called mim.exp,
    the MIM long form EXPRESS should be called mim_lf.exp as described in "Module Files"
  • Convert the EXPRESS to XML using the JScript express2xml.
  • Document the module, i.e. edit the file module.xml as described in "Module documentation"
  • Create the EXPRESS-G diagrams. If these are created as HTML imagemap, these should be converted to XML using the JScript imagemap2xml.
  • Convert the XML to HTML using the JScript xml2html.
