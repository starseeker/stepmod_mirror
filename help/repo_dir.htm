Repository directories

The module repository is organized as follows:
css/
Contains the CSS stylesheets. Note: these are not used for the ISO publication of the modules.

data/
XML data for the module repository.

data/basic
Contains boilerplate XML document fragments common to all modules.

data/modules
Contains directories for the modules.

data/modules/[module-name]
Each module is stored in its own directory with the same name as the module [module-name]. Each module directory contains files specific to the module including XML data, express-g graphics, etc.

data/modules/[module-name]/sys
Auto generated XML files for displaying the module.

data/resources
Contains directories of integrated resources schemas. Each schema is stored in its own directory with the same name as the schema.

dtd/
Contains the repository DTD files.

help/
Contains all the help files.

releasenotes/
Contains all the release notes.

utils/
Contains a set of utility scripts for creating modules, converting EXPRESS to XML, generating the HTML from XML.

xsl/
Contains everything pertaining to transforming the XML to HTML. This includes XSLT stylesheets.
