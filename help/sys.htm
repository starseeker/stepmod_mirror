Module SYS directory

This directory contains XML files that are automatically generated when a module is created. Each XML file corresponds to a clause of the module. The XML file has no content and is used to apply the XSL stylesheet relevant to the clause to the module.xml file. The resulting HTML file contains the contents of the clause.
