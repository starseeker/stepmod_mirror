EXPRESS XML elements

The XML representing EXPRESS objects should be generated automatically using the tool you use to create your EXPRESS-G. Your EXPRESS-G editor tool should export XML conforming to the EXPRESS Model Specification.

Avoid hand-editing EXPRESS XML elements. Otherwise, you may wind up with a module whose ASCII EXPRESS and EXPRESS-G are inconsistent.
